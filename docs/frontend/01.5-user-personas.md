# User Personas

## Overview

Before designing the sitemap, frontend architecture, pages or APIs, we first define the primary users of DeutschHub.

Each persona represents a different group of users with unique goals, motivations and learning journeys.

Instead of designing the product around features, DeutschHub is designed around people.

These personas will guide future decisions in:

- Product Design
- User Experience (UX)
- Frontend Architecture
- Backend APIs
- Content Strategy

---

# Persona 1 — Study Abroad Learner

### Description

This user plans to study or work in Germany in the future.

Their primary objective is to achieve the required German language proficiency (typically B1–B2 for vocational study or C1 for university) while preparing for life in Germany.

DeutschHub should become their long-term learning companion.

### Goals

- Learn German systematically.
- Reach the required language level.
- Understand German culture before arrival.
- Prepare study abroad documents.
- Build confidence before moving to Germany.

### Needs

- Structured language courses
- Learning roadmap
- Progress tracking
- German culture
- Study abroad guides
- Practical information about living in Germany

### Entry Points

- Google Search
- TikTok
- Facebook
- YouTube
- Recommendations from friends

### Success

DeutschHub succeeds when the learner is confident enough to continue their journey in Germany.

The platform is expected to accompany them from their first German lesson until they begin their new life abroad.

### Main Features

- Courses
- Learning
- Quiz
- Study Abroad
- Progress Tracking
- Certificates

### User Quote

> "I want to study in Germany, but I don't know where to start."

# Persona 2 — Traveler

### Description

This user plans to visit Germany for tourism or short-term travel.

Their focus is not mastering German, but understanding the country before the trip.

### Goals

- Discover beautiful places.
- Learn basic German expressions.
- Understand German culture.
- Prepare for the journey.

### Needs

- Travel guides
- German cities
- Nature
- Transportation
- Basic German phrases
- Cultural information

### Entry Points

- Google Search
- Instagram
- TikTok
- Travel Blogs

### Success

The user feels fully prepared before arriving in Germany and can travel with confidence.

### Main Features

- Explore Germany
- Culture
- Cities
- Nature
- Basic German Lessons

### User Quote

> "I'm visiting Germany next month. What should I know before I go?"

# Persona 3 — Germany Enthusiast

### Description

This user simply loves Germany.

They enjoy learning German, exploring the country's culture, history, architecture and daily life without having a specific academic or career goal.

This is one of the most valuable long-term users of DeutschHub.

### Goals

- Learn for enjoyment.
- Discover German culture.
- Improve German naturally.
- Explore interesting topics.

### Needs

- Articles
- Cultural stories
- German news
- Podcasts
- Videos
- Interactive learning

### Entry Points

- Social media
- Google
- Recommendations
- Existing learners

### Success

The user continuously returns to DeutschHub because it becomes their favorite place to explore Germany.

### Main Features

- Explore Germany
- Articles
- German Culture
- Courses
- Vocabulary
- Community (Future)

### User Quote

> "I enjoy learning about Germany because it inspires me."

# Persona 4 — Germany Resident

### Description

This user is already living in Germany.

Instead of learning the basics, they need reliable information for everyday life.

### Goals

- Stay informed.
- Understand regulations.
- Follow local news.
- Adapt to German society.

### Needs

- News
- Government information
- Daily life tips
- German laws
- Community resources

### Entry Points

- Google
- Existing users
- Recommendations

### Success

DeutschHub becomes a trusted information hub that helps them adapt to life in Germany.

### Main Features

- News
- Living in Germany
- Local Guides
- Community (Future)

### User Quote

> "I already live in Germany. I need trustworthy local information."

---

# Persona 5 — Casual Visitor

### Description

This user has no specific long-term goal.

They become curious after watching videos, reading articles or hearing something interesting about Germany.

They are potential future learners.

### Goals

- Explore Germany.
- Find interesting information.
- Learn something new.

### Needs

- Easy navigation
- Interesting articles
- Beautiful visuals
- Beginner-friendly content

### Entry Points

- Google
- TikTok
- Instagram
- Facebook
- Search engines

### Success

The visitor leaves with a positive impression and eventually becomes a registered learner.

### Main Features

- Home
- Explore Germany
- Featured Courses
- Search

### User Quote

> "I heard something interesting about Germany. I want to learn more."

---

# Summary

DeutschHub is designed around people, not pages.

Each persona represents a different journey, and every feature should exist to help at least one persona move closer to their goal.

This user-centered approach ensures that future frontend pages, backend APIs and learning experiences are built with clear purpose instead of unnecessary complexity.
